## Synopsis
<b>`ECHO message`</b><br>
This command outputs the given `message`.

## Return Value
Returns the message.

## Examples
```{.sh .copy .separator-dollar}
$ ECHO "Yuga Yuga"
```
```sh
"Yuga Yuga"
```
