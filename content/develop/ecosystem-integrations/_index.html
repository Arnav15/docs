--- title: Ecosystem Integrations weight: 570 --- YugaByte DB is driver compatible with Apache Cassandra and Redis clients with PostgreSQL coming soon. It can therefore work with the ecosytem of projects built around these APIs. This page lists some of the ecosystems that YugaByte DB can work with.
Apache Spark
Powerful open source processing engine built for fast analytics.
JanusGraph
Scalable graph API that uses Cassandra as it's underlying distributed database.
KairosDB
Fast time series metrics API that uses Cassandra as it's underlying distributed database.
