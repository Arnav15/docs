--- date: 2016-03-09T00:11:02+01:00 title: Quick Start weight: 10 --- The easiest way to get started with the YugaByte DB is to create a multi-node local cluster on your laptop or desktop.
1. Install YugaByte DB
Install the binary on Linux/Mac OS or use Docker to run on any OS of choice.
2. Create Local Cluster
Create a 3 node local cluster.
3. Test YugaByte CQL
Test YugaByte DB's CQL service.
4. Test YugaByte Redis
Test YugaByte DB's Redis service.
5. Run Sample Apps
Run pre-packaged sample apps.
 {{< note title="Note" >}} Running local clusters is not recommended for production environments. You can either deploy the [Community Edition] (/deploy/community-edition/) manually on a set of instances or use the [Enterprise Edition](/deploy/enterprise-edition/) that automates all day-to-day operations including cluster administration across all major public clouds as well as on-premises datacenters. {{< /note >}}
